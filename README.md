# henge (ヘンゲ)

> **形を変えて、道を切り拓け。新感覚「ヘンゲ」アクション！**

自分自身を「円・縦長・横長」の3つの形態に自由自在に変形させて進む、スタイリッシュなiOS向けアクションゲームです。一瞬の判断ミスが命取りになる、スリリングなステージ攻略が楽しめます。

## 📱 ゲーム概要

画面をタップして自らの形を瞬時に切り替え、多種多様なギミックが待ち受けるステージを突破していくアクションゲームです。プレイヤーの操作性と変形技術が試されます。

## ✨ 最大の特徴：3つの「ヘンゲ」

状況に応じて、以下の3つの形態を使い分けることがステージ攻略の鍵となります。

| 形態 | 特徴 | 主な用途 |
| :---: | :--- | :--- |
| **🔵 円** | 基本の形 | バランスの取れた動き |
| **▮ 縦長** | 縦にスマートな形 | 狭い隙間をすり抜けるのに最適 |
| **▬ 横長** | 横にワイドな形 | 低い天井や特殊なギミックを突破する鍵 |

## 🚧 多種多様なギミック（全10ステージ）

単純な壁だけでなく、プレイヤーの変形技術とタイミングを試す様々な障害物が登場します。

* **動く壁**：タイミングを合わせて形を変え、一瞬の隙間を狙って突き進みます。
* **追跡者**：背後から迫り来る障害物。
* **クランクゲート**：特定の形（ヘンゲ）でしか通ることができない、プレイヤーを惑わせる通り道。

## 🏆 やり込み要素と難易度設計

* **完全制覇を目指せ**：全ステージを通してノーミスでのクリアを目指すストイックなプレイが可能。
* **コイン集め**：ステージ内に隠されたコインをすべて集める収集要素を搭載。
